# Python Fundamentos

Repositório criado para registrar meus estudos em Python, reunindo exemplos práticos, exercícios e anotações dos principais conceitos da linguagem.

O objetivo deste projeto é servir como material de consulta rápida para revisões diárias e acompanhamento da minha evolução no aprendizado de programação.

---

## Conteúdos Estudados

### ✅ 01 - Variáveis

Principais conceitos:

* Tipos de dados (`str`, `int`, `float`, `bool`)
* Listas (`list`)
* Dicionários (`dict`)
* Impressão de valores com `print()`
* Verificação de tipos com `type()`
* Múltiplas atribuições
* Atualização de variáveis
* Operações matemáticas
* Concatenação de strings
* F-Strings
* Função `len()`
* Entrada de dados com `input()`

---

### ✅ 02 - Input

Principais conceitos:

* Recebimento de dados com `input()`
* Conversão de tipos (`int`, `float`, `str`)
* Utilização de entradas em cálculos
* Tratamento de erros com `try` e `except`
* Exceção `ValueError`

---

### ✅ 03 - Operadores

Principais conceitos:

* Operadores aritméticos (`+`, `-`, `*`, `/`, `//`, `%`, `**`)
* Operadores de comparação (`==`, `!=`, `>`, `<`, `>=`, `<=`)
* Operadores lógicos (`and`, `or`, `not`)
* Operadores de atribuição (`+=`, `-=`, `*=`, `/=`)
* Comparação de números
* Comparação de strings
* Comparação de valores booleanos

---

### ✅ 04 - Condicionais

Principais conceitos:

* Estrutura `if`
* Estrutura `if/else`
* Estrutura `if/elif/else`
* Condicionais com strings
* Operadores lógicos em condições
* Condicionais aninhadas
* Tomada de decisão baseada em regras
